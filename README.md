# Calculator Website

A simple, modern calculator web application that allows users to input values and perform calculations.

## Features

- Clean, modern UI with gradient design
- Multiple operations: Addition, Subtraction, Multiplication, Division, and Power
- Real-time result display
- Responsive design that works on all devices
- Input validation and error handling

## How to Use

1. Open `index.html` in your web browser
2. Enter two values in the input fields
3. Select an operation from the dropdown menu
4. Click "Calculate" to see the result

## Customization

You can easily customize this calculator by:

- **Adding more operations**: Edit `script.js` and add new cases in the switch statement
- **Changing the styling**: Modify `styles.css` to match your preferred design
- **Adding more input fields**: Update `index.html` and adjust the calculation logic in `script.js`

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Calculation logic and interactivity

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
